> Versão 1.0 (Implementação das 52 questões de fixação)

Este repositório contém as **52 questões de fixação em C** desenvolvidas para a disciplina **ESTRUTURAS DE DADOS**, do curso de **Engenharia de Computação** da UniJorge.  
Os exercícios abrangem desde operações básicas com números até problemas mais complexos envolvendo **estruturas, vetores, funções e ponteiros**.

---

## Conteúdo
Os exercícios estão divididos em duas partes:

- **Exercícios Básicos (1–29):**
  - Operações matemáticas simples
  - Estruturas condicionais
  - Laços de repetição
  - Tabuadas, médias e contagens
  - Problemas práticos com entrada de dados

- **Exercícios de Fixação (30–52):**
  - Cálculos geométricos (círculo, esfera, cilindro)
  - Conversões de temperatura
  - Progressões aritméticas
  - Estruturas de repetição e condicionais avançadas
  - Problemas aplicados em contexto real (salários, vendas, hotelaria, pesquisas)

---

## Tecnologias Utilizadas
- **Linguagem:** C  
- **Compilador:** GCC (MinGW-w64)  

---

## Como Compilar e Executar
Cada questão está implementada em arquivos separados (`questao1.c`, `questao2.c`, ..., `questao52.c`).  
Para compilar e rodar, utilize:

```bash
gcc questao1.c -o questao1 && ./questao1
gcc questao2.c -o questao2 && ./questao2
...
gcc questao52.c -o questao52 && ./questao52
```
